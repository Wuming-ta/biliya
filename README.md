## Biliya
###  如何本地测试
1. 安装tomcat
2. 项目打包 在控制台 mvn clean install
3. 移动mall-webapp target中的war包 放入tomcat中的webapps下
4. 运行bin文件夹中的startup.bat
5. 浏览器输入localhost:8080/{war包名字} 进入项目
